# Twitter_AutoPassExtract_for_Pythonista
This app extract only password from password tweet. This app runs on Pythonista(iOS app)
# Features
Copy the tweet URL of the target Twitter account and launch this app from the iPhone keyboard. The app extracts and pastes the password the moment the account tweets.

# Requirement
<PyPI>
* Tweepy
* requests_oauthlib
* oauthlib
<API>
* Twitter API(Application required)
# Installation
Rename the Twitter_AutoPassExtract_for_Pythonista folder to XAUTO. Place it in Documents (root) of the Pythonista app.
```bash
#<Stash on Pythonista>
git clone https://github.com/tsuchiya-i/Twitter_AutoPassExtract_for_Pythonista.git
mv Twitter_AutoPassExtract_for_Pythonista XAUTO
```
